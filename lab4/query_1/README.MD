# Анализ запроса №1

## Запрос на языке SQL

```sql
select Н_ОЦЕНКИ.ПРИМЕЧАНИЕ, Н_ВЕДОМОСТИ.ДАТА
from Н_ВЕДОМОСТИ
left join Н_ОЦЕНКИ on Н_ВЕДОМОСТИ.ОЦЕНКА = Н_ОЦЕНКИ.КОД
where (
    Н_ОЦЕНКИ.ПРИМЕЧАНИЕ > 'неявка'
    and
    Н_ВЕДОМОСТИ.ИД = 1457443
    and
    Н_ВЕДОМОСТИ.ИД > 39921
);
```
